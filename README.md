# aio-standalone

Standalone application server framework using asyncio.

## Installation

Grab it off github for now:

```
git clone https://github.com/kblin/aio-standalone.git
cd aio-standalone
pip install .
```

I'm hoping to make pypi packages available as soon as this stabilises.
